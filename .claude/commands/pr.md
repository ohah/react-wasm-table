Create a pull request for the current branch.

Instructions:

1. Check the current branch and its diff against the main branch
2. Review all commits that will be included in the PR
3. Create a PR with:
   - A concise title (under 70 characters)
   - A summary section with 1-3 bullet points
   - A test plan section
4. Push the branch if needed and create the PR using `gh pr create`
